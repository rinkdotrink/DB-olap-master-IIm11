\subsection{Aufgabenstellung}\label{Aufgabenstellung}
In diesem Projekt werden verschiedene Datenbankkonfigurationen bestimmten Anwendungsfällen zugeordnet. Jede Konfiguration ist für bestimmte Anwendungsfälle geeignet. Für jede Konfiguration wird die Performance gemessen. Um die Performance messen zu können, ist ein ERD für einen selbstgewählten Anwendungsfall anzufertigen. Die Tabellen des ERD sollen mit Testdaten gefüllt werden. Auf die gefüllten Tabellen können Queries gefahren werden. Die Zeit die ein Query braucht um auf einer bestimmten Konfiguration ausgeführt zu werden wird gemessen. Die Messergebnisse werden in einer Exceltabelle gesammelt und visualisiert.
