\section{Aufgabenstellung}\label{Aufgabenstellung}
In diesem Projekt wird jedem Anwendungsfall eine Konfigurationen zugeordnet. Jede Konfiguration ist für einen Anwendungsfall besonders geeignet. Für jede Konfiguration wird die Performance gemessen. Um die Performance messen zu können, ist ein ERD für einen selbstgewählten Anwendungsfall anzufertigen. Die Tabellen des ERD sollen mit Testdaten gefüllt werden. Dazu ist ein Datengenerator anzufertigen. Die Zeiten für die Inserts wird gemessen. Auf die gefüllten Tabellen werden Queries gefahren. Die Zeit die ein Query braucht, um auf einer bestimmten Konfiguration ausgeführt zu werden, wird gemessen. Die Messergebnisse werden in einer Exceltabelle gesammelt und visualisiert.
