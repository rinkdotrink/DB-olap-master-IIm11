\subsection{Datenbankabfragen}
\begin{figure}[htb]
	\centering
		\includegraphics[width=1.0\textwidth]{Steff/UC.png}
	\caption{Datenbankabfragen}
	\label{fig:AbfragenUC}
\end{figure}
Da unser Projekt einen Webshop imitieren sollte, haben wir unsere Anwendungsfälle so gewählt, wie sie auch in der Realität gefunden werden können. Sie könnten unter anderem Managern bei der Auswertung ihrer Geschäfte dienen. Das gesamte Spektrum an möglichen Abfragen abzudecken, hätte jedoch den Zeitrahmen extrem gesprengt. Deshalb wurde sich für die folgenden vier Abfragen entschieden:
\begin{itemize}
	\item Welches Produkt wurde wie oft gekauft?
	\item Wieviel Umsatz wurde von wem in einem bestimmtem Zeitraum generiert?
	\item Wie viele Kunden haben an einem bestimmten Zeitpunkt bestellt?
	\item Wer hat wieviel an den gegebenen neun Tagen umgesetzt?
\end{itemize}
Diese Abfragen haben unterschiedliche Komplexitäten. So müssen bei der zweiten Abfrage sämtliche Tabellen einbezogen werden, während bei der ersten Abfrage nur zwei Tabellen involviert sind.
Auf diese Weise soll ein möglichst breites Spektrum an Abfragen getestet werden, um sicher zu stellen, dass die Versuche so aussagekräftig wie möglich werden.
